# 学生就业管理系统 python1223

## 项目概述

学生就业管理系统是一个基于Python Django框架开发的系统，旨在帮助学校或教育机构管理学生的就业信息。本项目包含完整的源代码和开发文档，便于用户进行二次开发或项目维护。

## 技术栈

- 后端：Python 3, Django
- 前端：jQuery
- 数据库：MySQL

## 功能模块

系统主要包括以下几个功能模块：

1. 学生信息管理
2. 企业信息管理
3. 招聘信息发布
4. 求职信息管理
5. 数据统计与分析

## 系统角色

- 系统管理员
- 学生
- 教师/就业指导老师
- 企业HR

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x

## 部署步骤

1. 安装Python 3和MySQL
2. 克隆项目代码到本地
3. 导入MySQL数据库
4. 安装依赖库（根据`requirements.txt`）
5. 配置`settings.py`中的数据库连接信息
6. 运行`python manage.py runserver`启动开发服务器
